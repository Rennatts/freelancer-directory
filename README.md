## freelancer-directory
A full-stack freelancer directory 


## DATABASE
MONGODB

## BACK-END
NESTJS

## FRONT-END
REACT

## STYLE
TailwindCss

## STATE MANAGEMENT
Redux Tooltip



## HOW TO RUN

1- Run the server

```
cd server
npm install

npm run start:dev
```
2- Run the client


```
cd client
npm install

npm start
```

### To see the swagges
http://localhost:3000/api
